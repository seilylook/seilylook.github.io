# Mathematics Standard Deviation


표준 편차(Standard Deviation)는 데이터 세트의 분포를 나타내는 통계적 척도입니다. 이는 데이터가 평균으로부터 얼마나 떨어져 있는지를 나타내며, 데이터의 변동성을 측정하는 데 사용됩니다. 표준 편차가 작을수록 데이터가 평균에 가깝게 모여 있음을 의미하고, 표준 편차가 클수록 데이터가 평균에서 멀리 떨어져 있음을 의미합니다.

### 표준 편차의 계산 방법
표준 편차는 다음과 같은 단계로 계산됩니다.

1. **평균(산술 평균)**을 계산합니다:
   
    $$
   
   \mu = \frac{1}{N} \sum_{i=1}^{N} x_i
   
   $$

   여기서 \(N\)은 데이터 포인트의 개수이고, \(x_i\)는 각 데이터 포인트입니다.

2. **분산(Variance)**을 계산합니다:

   분산은 데이터가 평균으로부터 얼마나 떨어져 있는지의 제곱의 평균입니다.

   $$
   \sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2
   $$

3. **표준 편차**를 계산합니다:

   표준 편차는 분산의 제곱근입니다.
   
   $$
   \sigma = \sqrt{\sigma^2} = \sqrt{\frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2}
   $$

### 표준 편차의 특징
- **단위**: 표준 편차의 단위는 원 데이터의 단위와 동일합니다. 이는 분산과의 주요 차이점 중 하나입니다.
- **평균으로부터의 거리**: 표준 편차는 평균으로부터 데이터가 얼마나 떨어져 있는지에 대한 정보를 제공합니다.
- **정규 분포**: 데이터가 정규 분포를 따를 경우, 표준 편차는 정규 분포의 폭을 나타내는 중요한 지표입니다. 예를 들어, 정규 분포에서 약 68%의 데이터는 평균 ± 1 표준 편차 내에 존재합니다.

### 예제
예를 들어, 데이터 세트가 [2, 4, 4, 4, 5, 5, 7, 9]라면, 다음과 같이 표준 편차를 계산할 수 있습니다:

1. **평균**: \(\mu = \frac{2 + 4 + 4 + 4 + 5 + 5 + 7 + 9}{8} = 5\)

2. **분산**:
   $$
   \sigma^2 = \frac{(2-5)^2 + (4-5)^2 + (4-5)^2 + (4-5)^2 + (5-5)^2 + (5-5)^2 + (7-5)^2 + (9-5)^2}{8} 
            = \frac{9 + 1 + 1 + 1 + 0 + 0 + 4 + 16}{8} 
            = \frac{32}{8} = 4
   $$

3. **표준 편차**:
   $$
   \sigma = \sqrt{4} = 2
   $$

따라서, 이 데이터 세트의 표준 편차는 2입니다.

### 실생활에서의 예
표준 편차는 다양한 분야에서 사용됩니다. 예를 들어, 금융에서는 주식의 가격 변동성을 나타내기 위해, 품질 관리에서는 제품의 균일성을 평가하기 위해 사용됩니다. 교육에서는 시험 점수의 변동성을 측정하여 학생들의 성취도를 평가하는 데도 사용됩니다.
