# Krypto


# Introduction

`React`와 `Solidity`를 사용해 `Ethereum` 거래 서비스 구현.

## Tech stack

- Front-end

  - React
  - Tailwindcss
  - ethers (version 5)
  - vite

- Smart Contract

  - hardhat
  - chai
  - ethers (version 5)
  - ethereum-waffle(version 4)
